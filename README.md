# Portfolio
This repository contains my personal portfolio website, built using HTML, CSS, and JavaScript, to showcase my profile as an AI/ML Python Developer. The portfolio highlights my background, technical skills, work experience, projects, and research in a clean, modern, and interactive design. It features a responsive layout, smooth scrolling, 3D card hover effects, glassmorphism UI, and animated elements to deliver a professional user experience across devices. The website includes dedicated sections for projects, AI/ML skills, LLM systems, research publications, and direct links to my GitHub, LinkedIn, and other professional platforms, serving as a central hub for recruiters and collaborators to explore my work and expertise.
